    ## Удобные алиасы в терминале

Список того, что добавлено и как работает:

**Основные алиасы**

`fileviev` — применяет цвета из wal (pywal) к терминалу, очищает экран и открывает файловый менеджер ranger.

`themes` — запускает скрипт для переключения тем оформления (hypr.themes-utiliti-v2).

`ls / ll / la / l / l.` — используют `exa` вместо стандартного `ls` с иконками и цветами.

`c` — быстрая очистка экрана (clear).

`py` — запуск Python 3.

`m` — запуск dotify с отключённой композицией WebKit (для работы dotify).

`zapret` — запуск скрипта для обхода блокировок Discord и YouTube.

`update` — обновление системы через pacman (sudo pacman -Syu).

**Git алиасы (сокращения)**

`g` — git  
`ga` — git add  
`gaa` — git add --all  
`gc` — git commit -v  
`gcm` — git commit -m  
`gco` — git checkout  
`gcb` — git checkout -b  
`gst` — git status  
`gl` — git pull  
`gp` — git push  
`gpf` — git push --force-with-lease  
`gd` — git diff  
`gds` — git diff --staged  
`glog` — git log --oneline --graph --decorate  
`gundo` — git reset --soft HEAD~1 (отмена последнего коммита, изменения остаются в staged)

**Открытие файлов по расширению** (достаточно набрать имя файла, и он откроется в нужной программе)

`.txt` / `.py` / `.js` / `.sh` / `.css` / `.scss` / `.rs` / `.qml` / `.conf` / `.html` — открываются в редакторе `zed` (или `nvim` для `.md`).

`.jpg` / `.png` — открываются в `feh`.

`.pdf` — открываются в `zathura`.

`.md` — открываются в `nvim`.
